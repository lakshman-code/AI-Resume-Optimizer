## ğŸ› ï¸ Troubleshooting

### 1. MongoDB Connection Error (`ECONNREFUSED`)

If you see `connect ECONNREFUSED 127.0.0.1:27017` in the server logs:

- **Local MongoDB:** Ensure your MongoDB service is running (`mongod` in terminal or check Windows Services).
- **Remote MongoDB:** Update the `MONGO_URI` in your `.env` file with your connection string from MongoDB Atlas.

### 2. OpenAI SDK Errors

The project uses the latest OpenAI Node.js SDK (v4+). Ensure your `server/package.json` reflects `"openai": "^4.x.x"`.

### 3. Client Import Errors (`Failed to resolve import "axios"`)

If the client fails to start because of missing dependencies:

```bash
cd client
npm install axios
```
